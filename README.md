# Minimal Next.js Two-Page App

A simple, beautifully designed Next.js application with two pages.

## Pages

- **Home** (`/`) - Landing page with hero section
- **About** (`/about`) - About page with feature cards

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14 (App Router)
- React 18
- CSS (Custom Properties + Animations)
